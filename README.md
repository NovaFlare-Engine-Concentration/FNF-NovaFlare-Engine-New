## 简介
## Introduction

**Friday Night Funkin' - NovaFlare Engine** 现在是一个基于 [Heaps](https://heaps.io/) 游戏引擎的现代化重写版本。原项目基于 Haxe 和 Psych Engine，现已转型为使用 Heaps 以提供更好的性能、跨平台支持和模块化架构。

**Friday Night Funkin' - NovaFlare Engine** is now a modern rewrite based on the [Heaps](https://heaps.io/) game engine. The original project was based on Haxe and the Psych Engine, and has now transitioned to using Heaps to provide better performance, cross-platform support, and a modular architecture.

该引擎最初用于 VS Camellia fanmade 模组，后来扩展为支持自定义模块，专注于优化和性能，为玩家提供最佳体验。

The engine was originally used for the VS Camellia fanmade mod and has since expanded to support custom mods, focusing on optimization and performance to provide the best experience for players.

## 功能特性
## Features

- **高性能渲染**: 利用 Heaps 的高效 2D/3D 渲染引擎

- **High-Performance Rendering**: Utilizes Heaps' efficient 2D/3D rendering engine

- **模块化支持**: 支持自定义模组和扩展

- **Modular Support**: Supports custom mods and extensions

- **跨平台兼容**: 支持 Windows、macOS、Linux、Web 和移动平台

- **Cross-Platform Compatibility**: Supports Windows, macOS, Linux, Web, and mobile platforms

- **优化体验**: 专注于帧率稳定和资源管理

- **Optimized Experience**: Focuses on stable framerates and resource management

- **社区驱动**: 基于开源社区贡献

- **Community Driven**: Built on open-source community contributions
